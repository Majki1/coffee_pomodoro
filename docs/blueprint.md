# **App Name**: Coffee Time

## Core Features:

- Work Timer: Set a timer for a work period, visualized by a coffee cup slowly emptying.
- Rest Timer: Set a timer for a rest period, visualized by a coffee cup slowly filling up.
- Session Customization: Allow users to customize the duration of work and rest periods.
- Notification Sounds: Play different sounds for the start and end of work and rest periods.
- Session Logging: Keep a history of completed pomodoro sessions.
- Inspirational Quotes: Display an inspirational quote about productivity during rest periods.

## Style Guidelines:

- Primary color: Deep brown (#6F4E37) evoking coffee and warmth.
- Background color: Light beige (#F5F5DC) creating a cozy atmosphere.
- Accent color: Soft orange (#FFB347) highlighting active timer.
- Body and headline font: 'PT Sans' for a friendly and modern feel.
- Simple line icons representing timer settings and session history.
- Centered layout with large, readable timer display.
- Smooth transitions for timer updates and quote changes.
